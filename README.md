# first initialize the git
```
git init
```

# To add the file
```
git add abc.txt
```

# To add all the file
```
git add .
```

# To commit the changes
```
git commit -m "this is my first commit"
```

# To push all changes after commit
```
git push
```
# To pull all changes from git
```
git pull
```
